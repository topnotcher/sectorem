sectorem
========

Async Python client for the Charles Schwab API.

Sectorem wraps the Schwab Trader and Market Data APIs behind a clean,
Pythonic interface. It handles OAuth2 authentication, token lifecycle
management, and presents API responses as typed dataclasses rather than
raw JSON.
